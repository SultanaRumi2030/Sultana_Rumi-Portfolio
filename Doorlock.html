Automatic DoorLock System using Arduino

About This Project

This Arduino-based automatic door lock system enhances home security by using modern authentication methods such as RFID, password entry, or Bluetooth control. The system is built on Arduino Uno and connected with servo motors to manage the locking mechanism, offering both security and convenience.

Secure Authentication

Supports RFID, Bluetooth, and password input for flexible security options.

Arduino Powered

Built on Arduino Uno, ensuring smooth integration with sensors and motors.

Smart Home Ready

Can be integrated with IoT systems for remote access and monitoring.
